# Comment Script

This file contains a simple shell script demonstrating single-line and multi-line comments in Bash.

```bash
#!/bin/bash
#This is called shebang.

#Multi line comment
<< comment
 This shebang is given so that os will understand which 
scripting language to use. As there are different kind of scripting.
Eg. Shell, Bash, Ksh 
comment

#This statement will be printed
echo "Single and multi line comment"
```

## What it does

- Shows the shebang line used to indicate the shell interpreter.
- Demonstrates a multi-line comment block using `<< comment` and `comment`.
- Prints a message to the terminal.
